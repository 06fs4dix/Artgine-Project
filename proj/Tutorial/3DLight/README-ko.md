3D 환경에서 다양한 **라이팅**과 **그림자**, **스카이박스**를 확인할 수 있는 샘플입니다.

## 사용 방법

앱의 **Help** 버튼을 열고 각 모드를 선택하여 동작을 확인하세요.

- **ShadowPlane**: Y축에 정렬된 평면(Plane)에 그림자를 투영해 표시합니다.
- **Forward**: 포워드 렌더링으로 **섀도 맵**을 생성하고 적용합니다.
- **Deferred**
  - **Single**: **Half‑Lambert + Phong** 조명 모델을 사용합니다.  
    **패럴랙스(Parallax)** 렌더링으로 깊이감을 추가합니다.
    여러 렌더 패스를 수행하고, 각 결과를 텍스처로 **베이크**하여 사용합니다.
  - **Multi**: 한 번의 렌더링으로 여러 속성을 동시에 출력(G-buffer 스타일).  
    **PBR**을 사용하며, 스카이박스 기반 **환경광(ambient)** 을 추가합니다.
